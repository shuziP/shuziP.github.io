明日方舟同人创作分析报告¶

前言¶

这份数据分析报告目的是探究明日方舟同人创作作品，以同人二次创作的质量与玩家活跃度来分析该游戏自公测以来开放的游戏活动质量。

《明日方舟》是由鹰角网络开发的一款国产战略经营类游戏，于2019年5月1日公测。是TapTap游戏平台在2019年年度最佳游戏/最受玩家喜爱游戏/最具影响力国产游戏。 相比于其他手游，《明日方舟》的一大特点在于她是一款二次元手游，依靠二次元核心用户口碑宣传，二次创作吸引更多的玩家“入坑”游戏，B站则是玩家们二次创作发布的聚集地。

所以对B站《明日方舟》稿件的追踪与数据分析，以评判该手游的活动质量是有必要的。

B站的《明日方舟》相关的稿件有着更多的种类，游戏攻略，手书视频，同人音乐，背景考究等...

在游戏加入新元素（开启新副本，加入新的卡池）时会引起更多的二次创作， 优质的二次创作进而吸引更多的玩家，社区活跃度上升的除了能为游戏厂商带来更多的收益的同时，也给玩家带来更多元化的游戏体验。

所以，分析同人创作的活跃度可以是评价《明日方舟》活动好坏的一大指标。

这份报告对明日方舟开服一周年以来，B站质量最高的Top1000的稿件（以收藏量为标准）进行数据分析，对游戏中对各大活动时间线对应对投稿质量，以及玩家们关心对关键词进行分析，得出报告。

word_list = []
for i in data['标题']:
    word_list.append(i)

# 文本预处理
pattern = re.compile(u'\t|\n|\.|-|:|;|\)|\(|\?|"') # 定义正则表达式匹配模式
string_data = re.sub(pattern, '', string_data) # 将符合模式的字符去除

# 文本分词
seg_list_exact = jieba.cut(string_data, cut_all = False) # 精确模式分词
object_list = []
remove_words = [u'【', u'】', u' ', u',', u'！', u'，', u'（', u'）', u'/', u'？', u'的', u'了', u'你', u'的'
               , u'《', u'》', u'：', u'—', u'是', u'!', u"'", 
               ] # 自定义去除词库


for word in seg_list_exact: # 循环读出每个分词
    if word not in remove_words: # 如果不在去除词库中
        object_list.append(word) # 分词追加到列表

# 词频统计
word_counts = collections.Counter(object_list) # 对分词做词频统计
word_counts_top20 = word_counts.most_common(20) # 获取前10最高频的词
print (word_counts_top20) # 输出检查

结果及可提共的建议¶

从上述图表分析来看，在激发高质量二次创作稿件这一衡量纬度上，明日方舟最成功的任务是2019年07月09日开始的主线“靶向药物”开启纪念限时活动

同时，2019年05月30日的 骑兵与猎人与同年08月15日的 SideStory「火蓝之心」黑曜石节也有这不错的反响

在2020年上半年里，明日方舟最为成功的活动为 洪炉示岁。

后续的活动可以总结上述成功的案例继续策划进行。

在词频统计方面，动画、手术、MMD是一大二次创作的热门，目前，明日方舟已经多次在B站上举办视频创作大赛，但此类比赛是任何稿件都可以参与都项目 在未来，也许可以在手书、动画、MMD上单独设置赛道，举办特别的比赛。
